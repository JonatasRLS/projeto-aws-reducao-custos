## Descrição do Projeto

O projeto de implementação de ferramentas foi dividido em 3 etapas, cada uma com seus objetivos especí­ficos. A seguir, serão descritas as etapas do projeto:

Etapa 1:

- Amazon EC2
- Hospedagem de servidores em nuvem
- Mudança de servidores físicos locais da empresa para instâncias EC2, isso reduziria custos com infraestrutura na empresa, como: diminuição do consumo de energia, manutenção de hardware e permitiria escalabilidade sob demanda caso necessário.

Etapa 2:

- Amazon S3
- Armazenamento de objetos em nuvem
- Substituição de servidores físicos para armazenamento de documentos coorporativos, backups e arquivos do sistema para a utilização do amazon S3, reduzindo novamente, os custos de infraestura para manter os servidores locais.

Etapa 3:

- AWS lambda
- Execução de funções sem necessidade de servidores
- Implementação de funções Lambda para automação de rotinas internas, como processamento de arquivos, geração de relatórios e validação de dados, eliminando a necessidade de servidores ativos em tempo integral.
